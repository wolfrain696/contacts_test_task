# Contacts Test Task 

Для запуска проекта необходимо установить зависимости для сервера и клиента
### `cd .\client\`
### `npm i`
### `cd .\server\`
### `npm i`

Далее нужно запустить сервер и клиет

* Для сервера:
### `cd .\server\`
### `npm start`

* Для клиета:
### `cd .\client\`
### `npm start`
